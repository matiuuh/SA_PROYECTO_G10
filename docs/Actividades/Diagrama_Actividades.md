# Diagrama de Actividades

El diagrama a continuación muestra las actividades principales del sistema:

1. Registro del sistema
2. Inicio de sesión
3. Selección de perfil 
4. Selección de plan de suscripción
5. Proceso de pago
6. Acceso al contenido
7. Busqueda y filtrado de contenido 
8. Reproducción de contenido y guardado de progreso
9. Puntuación de contenido

![Diagrama de Actividades](./img/Proyecto-SA-Junio2026-Vista_de_Actividades.drawio.svg)

Este diagrama representa el flujo de actividades que los usuarios pueden realizar dentro del sistema, desde el registro hasta la interacción con el contenido disponible. Cada actividad está conectada para mostrar la secuencia lógica de acciones que los usuarios pueden seguir.
